Workshop: Cutiepa2D.js

Thursday, 2/13 @ 5-6:30pm, Location TBD

Greyson Parrelli will give a presentation on how to use Cutiepa2D.js, a framework built on top of CreateJS that allows for rapid game development. No javascript knowledge is necessary, but you should understand basic programming concepts (variables, functions, etc). The first 30-45 minutes will be lecture-style, and the remainder of the time will be Q&A and "office hours".

Workshop: Construct 2

Thursday, 2/20 @ 5-6:30pm, Location TBD

Michael Green and Sheyn Denizov will be giving a presentation on Construct 2, a Windows program that allows you to use drag-n-drop with a visual logic editor to create games quickly and easily without any programming. Construct 2 is free for making Windows 8 games, and can also makes games for almost any other mobile platform if you pay a licensing fee. The first 30-45 minutes will be lecture-style, and the remainder of the time will be Q&A and "office hours".
